Allow users to submit voice comments on your WordPress posts.

== Description ==
This plugin enables voice recording and playback for comments on your WordPress site. Users can record audio directly in their browser, and the audio will be attached to their comment.

== Installation ==
1. Upload the `voice-comments` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
